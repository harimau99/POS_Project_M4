Login
In order to gain access into this system and use the features, you would need an authorized username and password. Certain features are reserved for managerial positions only. To login, click on Login under Access Control in the Main Menu, enter your username and password and click OK.
Change Password
To change your password, click on Change Password under Access Control in Main Menu, enter the required fields and click OK button to save.
Logout
In order to logout, click Logout under Access Control in Main Menu.
Exit
To exit the system, click Exit under Access Control in Main Menu.
